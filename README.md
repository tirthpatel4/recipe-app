# 🍳 RecipeApp2025

RecipeApp2025 is a cross-platform mobile application built using .NET MAUI that allows users to browse and view recipes in a clean, easy-to-navigate interface.  

This project was developed as a **group project through the Collaborative Software Development Club**, where we worked together to design, build, test, and refine the application from start to finish.

The goal of this project was to create a functional mobile app that demonstrates strong fundamentals in app structure, UI design, navigation, and collaborative software development.

---

## Features

- Browse a list of recipes  
- View detailed recipe pages  
- Smooth navigation between screens  
- Clean and mobile-friendly layout  
- Cross-platform support (Windows, Android, iOS)  

---

## How to Run the Project

1. Clone the repository  
2. Open the solution file in Visual Studio 2026  
3. Select your desired target device:
   - Windows Machine  
   - Android Emulator  
   - iOS Simulator (if configured)  
4. Click **Run (▶)**
